# Crypto Intelligence Pro

🚀 A next-generation crypto trading decision support system built with Streamlit.

## Features
- 📊 Live candlestick charts with RSI + Moving Averages
- 📰 News sentiment analysis (VADER NLP)
- 🎤 Political & macro shock detector
- 🛡 Risk management — Stop Loss & Take Profit guidance
- 📧 Email alert system
- 📡 **Offline mode** — works without internet using demo data

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Data Sources
- Yahoo Finance (price data)
- CoinDesk, CoinTelegraph, Decrypt, CryptoSlate, Reuters, BBC, CNBC (news)
